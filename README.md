# War Simulator

観戦型のオート戦争シミュレーションゲームプロトタイプです。
架空のマップ（ボロノイ図を用いた動的な領土表現）上で、複数の国家が自動的に戦争・併合を繰り返します。

## Demo

https://ateliee.github.io/war-simulator/

## プロジェクト構成

* **Godotバージョン**: Godot 4.6
* **ディレクトリ構成**:
  * `scenes/`: UIやゲームオブジェクトのシーンファイル
  * `scripts/`: ゲームロジック（GDScript）
  * `shaders/`: 領土を描画するためのボロノイシェーダー
  
## システム解説
* **兵力の増加**: 時間経過とともに各国の兵力が自動増加します。
* **交戦と被害**: 国境を接している敵対国同士は、毎フレーム自動的にダメージ（兵力の減少）を与え合います。
* **併合**: 兵力が 0 以下になった国は滅亡し、隣接する敵国に併合され、領土の色が書き換わります。

## 起動方法
Godot 4.6エディタでこのディレクトリを開き、実行ボタン（Play）を押すか、以下のコマンドで実行できます。

```bash
godot --path .
```
